我需要做 3 个 Tab 页面：首页、剧场、我的。

首页：
1. 可以上下拖动的视频流（可以理解为纵向的PageView），每次拖动都是切换到下一个短剧系列。
2. 顶部无标题栏，整个页面（除了底部Tab区域）都是视频区域，视频填满屏幕宽度。
3. 首页底部覆盖一个半透明悬浮条，填满屏幕宽度，显示“观看完整短剧·全xx集”，点击后进入新的播放页面（播放进度需同步），同样是一个可以上下拖动的视频流，在这个页面上下拖动会切换同一个短剧的不同集数。这个页面右上角有一个“倍速”按钮，点击可以选择（0.75、1.0、1.25、1.5、2.0、3.0）的播放速度。
4. 当当前短剧的比例为大于等于4:3时，在视频的下面显示一个按钮“全屏观看”，点击之后进入新的横屏全屏播放页面（播放进度需要同步）。

剧场：
1. 顶部无标题，有一个搜索框，点击进入搜索页面（暂不实现功能）。
2. 顶部下方是类目（也是Tab / Page 形式，可以左右滑动切换类目），有“找剧、漫剧、电影、电视剧”类目。
3. 下面整个区域是可上下自由滚动的短剧封面（2列），点击封面会进入「首页」-3 的同一个页面。

我的：
1. 包含登录按钮（无需实现）
2. 下方是播放历史：显示已看过的短剧和进度。
