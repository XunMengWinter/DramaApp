# DramaApp

一款使用 **SwiftUI** 开发的短剧 App Demo，聚焦短剧类应用的核心页面与交互实现，适合用于学习：
- 短剧首页推荐流
- 剧场列表展示
- 个人中心页面搭建
- SwiftUI 多 Tab 结构组织
- 短剧类 App 的基础 UI 设计与页面拆分

## 项目介绍

DramaApp 是一个面向初学者的短剧应用示例项目，使用 SwiftUI 构建，主要用于演示短剧 App 的基础产品结构与界面实现思路。  
项目包含首页、剧场、我的等核心页面，适合作为 SwiftUI 实战练习、课程演示项目，以及 AI 编程辅助开发示例。
